# Guia de Contribuição

Obrigado por querer contribuir com o Management Fleet!

## Como contribuir

1. Faça um fork do projeto
2. Crie uma branch para sua feature (`git checkout -b feature/nome-feature`)
3. Commit suas alterações (`git commit -m 'Minha feature'`)
4. Push para o branch (`git push origin feature/nome-feature`)
5. Abra um Pull Request

## Boas práticas
- Escreva commits claros e objetivos
- Sempre escreva testes para novas funcionalidades
- Documente endpoints e regras de negócio
- Siga o padrão de código do projeto (Java, Spring Boot)
- Revise Pull Requests de outros contribuidores

## Dúvidas ou sugestões?
Abra uma issue ou participe das discussões no GitHub!
